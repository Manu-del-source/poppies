# Poppies Premium Redesign

A world-class, premium digital experience for Poppies Restaurant, Nairobi. This redesign transforms the original website into a luxury-focused, cinematic platform while maintaining the brand's core identity.

## New Features & Enhancements

### 1. Visual & Aesthetic Upgrades
- **Editorial Typography**: Implemented 'Playfair Display' for high-end headings and 'Inter' for clean body text.
- **Refined Dark Theme**: Deep charcoal palette (#0a0a0a) with vibrant orange accents (#ff5e00) and glow effects.
- **Glassmorphism UI**: Frosted glass navigation and components for depth and modern feel.
- **Cinematic Imagery**: High-resolution, artistic food and interior photography.
- **Sophisticated Spacing**: Increased whitespace and refined visual hierarchy for a "breathable" luxury experience.

### 2. Interactive & Motion Enhancements
- **Fluid Scroll Experience**: Enhanced "scrolled" navigation states and smooth anchor transitions.
- **Advanced Reveal Animations**: Intersection Observer-based staggered reveals with subtle scaling and parallax.
- **Interactive Gallery**: Modern grid with artistic overlays and smooth hover transitions.

### 3. Redesigned Components
- **Premium Nav**: Minimalist glassmorphism bar with scroll-triggered styling.
- **Luxury Menu Cards**: Refined pricing visibility, elegant borders, and clean typography.
- **Editorial Reviews**: Minimalist guest feedback section prioritizing typography.
- **Modern Contact Grid**: Structured layout for location, hours, and direct WhatsApp ordering.

## Technologies Used
- HTML5 / CSS3 (Vanilla)
- Google Fonts (Playfair Display, Inter)
- JavaScript (Intersection Observer, Scroll APIs)
- Unsplash (High-End Visuals)

Redesigned with Gemini CLI.
