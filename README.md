# Poppies - Urban Dining in Nairobi

Poppies is a modern, mobile-first restaurant website designed for a contemporary urban dining experience in Nairobi, Kenya.

## Features
- **Dark-Themed Aesthetic**: High-impact design with vibrant orange accents.
- **Mobile-First**: Fully responsive for the best experience on smartphones.
- **WhatsApp Integration**: Direct ordering via WhatsApp links.
- **Interactive Menu**: Smooth animations and clear pricing in KES.
- **Fast Performance**: Lightweight single-page architecture.

## Menu Highlights
- **The Poppies Burger**: Juicy beef with secret sauce.
- **Peri Peri Quarter Chicken**: Flame-grilled perfection.
- **Loaded Nairobi Fries**: Masala chips with all the toppings.
- **Classic Nyama Choma**: The ultimate local favorite.

## Tech Stack
- HTML5
- CSS3 (Custom Properties, Flexbox, Grid)
- Vanilla JavaScript (Intersection Observer API)

## Location
Kenyatta Avenue, CBD, Nairobi, Kenya.
