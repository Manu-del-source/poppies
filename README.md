# Poppies "Urban Soul" Redesign (v3)

A world-class, premium digital experience for Poppies Restaurant, Nairobi. This version represents a complete architectural overhaul, moving from a static page to a modern, interactive React application.

## New Features & Enhancements

### 1. Rhythmic Layout & Navigation
- **Sticky Glassmorphism Navbar**: Tracks scroll progress and active sections.
- **Scroll Progress Indicator**: Visual line showing the user's journey through the site.
- **Alternating Sections**: Dynamic layout rhythm (Image-Left/Text-Right) to eliminate the "long page" feel.

### 2. Immersive Interactivity
- **Horizontal Meal Slider**: Drag/Swipe interactive carousel for featured dishes.
- **State-Driven Menu**: Real-time filtering by category (Burgers, Grill, Cocktails) with smooth Framer Motion transitions.
- **Custom Glow Cursor**: A refined, trailing ambient light that interacts with buttons and links.
- **Magnetic Buttons**: Physics-based button reactions for a high-end tactile feel.

### 3. Luxury UI Components
- **Optimized Rendering**: Lazy loading and "Blur-up" transitions for all high-res cinematic images.
- **Premium Reservation Modal**: Fully integrated glassmorphism booking experience for date/time/guest selection.
- **Chef's Special Widget**: Dynamic floating promo for real-time engagement.
- **Atmosphere Gallery**: Grayscale-to-color interactive grid capturing the urban soul of Nairobi.

### 4. Professional Tech Stack
- **React (Vite)**: For lightning-fast performance and modular component architecture.
- **Tailwind CSS**: Utility-first luxury styling.
- **Framer Motion**: Award-winning animation engine for fluid, natural movement.
- **Lucide React**: Modern, minimalist iconography.
